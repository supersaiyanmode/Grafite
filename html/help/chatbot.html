Getting Started

There are three, very obvious steps in using Grafite (Duh! :P)
 

Register

Very much like you would add your friend, you need to add GrafiteBot (with gtalk id as grafiteapp@appspot.com) to your friends list.
Once you are done adding Grafite, you need to register. Registration is simple. Just type register your-new-password. If I choose a password qwe!@#, then I would type 

register qwe!@#
in the chat window, to register myself.
[This crappy registration process will change in some time :P]

Authorise

The basic idea behind Grafite is to fetch information and update your status from all the social networking sites, at one go. Obviously, you need to authorise Grafite to fetch details on your behalf. Grafite uses OAuth Protocol for the same. To authorise a certain service, click on the authorise link that appears in your home page or the chat window. If its the chat window that you are using, you have to type 

status
and send it accross to grafite, to get the URL. More details below.

Fetch status Info & Update it

Grafite Chat bot can fetch details for you from all the services you have authorised it to fetch from. Just type "status" in the chat window. Grafite fetches data from the servives, if it can, else it will give you the links to authorise itself.
You can update a status of all the social networking sites at one go by issuing the "update" command. Like, if I had to update my status as "Hey, I am trying out Grafite", I would type, in the chat window: 

update Hey, I am trying out Grafite

Chat Bot Commands

Grafite Chat bot doesn't really understand human languages. Speak anything that it doesn't understand and it responds with a random human-like I-Dont-Understand-Reply.
BTW, the commands that GrafiteBot understands are summarised below.

  • help
    Displays a compact list of all commands, and a link to this page
  • status
    Fetches the number of comments/likes/retweets on your lastest status.
    In case you haven't authorised Grafite for a certain service, the link to authorise the same will be displayed. This is one way how new users can quickly authorise Grafite.
  • update <your status here minus these triangular brackets>
    This will post the specified update to all the connected services at one go.
  • home
    This command gives you the link to access your homepage on the website here, without logging in. Very simple.
  • shorten http://the.url.to/be/shortened
    Issuing this command will shorten the given URL using the goo.gl service.
  • remind in <time-value-integer> (minutes|hours|days|weeks) <reminder text>
    Like a friend pinging you up and reminding you about something. Grafite pings you. Eg: 
    remind in 3 hours call up project report pending
